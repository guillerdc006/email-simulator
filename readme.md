# 📧 Email Simulator

A simple console-based email simulator built with Python using Object-Oriented Programming (OOP).

This project was developed as part of the Python Certification course from FreeCodeCamp.

---

## 🚀 Features

* Create users
* Send emails between users
* Inbox management
* Read emails
* Delete emails
* Email status tracking (Read / Unread)
* Timestamp generation

---

## 🛠️ Technologies Used

* Python 3
* Object-Oriented Programming (OOP)
* `datetime` module

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/email-simulator.git
```

2. Navigate to the project folder:

```bash
cd email-simulator
```

3. Run the program:

```bash
python email_simulator.py
```
